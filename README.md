# Node.js Free tools code

I am learning and making free web tools using nodejs 

## Run Locally

Clone the project

```bash
  git clone https://github.com/mandeepsng/nodejs-tutorials
```

Go to the project directory

```bash
  cd nodejs-tutorials
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
